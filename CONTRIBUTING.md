# Contributing to RodinWorks Open Validation Hub

Thank you for helping advance open validation in dental 3D printing!

## What You Can Contribute
- Resin → Printer validation data  
- Print profiles (.sol, .pwmx, .ini)  
- STL test models  
- Accuracy reports (PDF, CSV, MD)  
- Print photos  
- Exposure tables  
- Notes on failures and anomalies  

## How to Submit a Validation

1. Fork this repository  
2. Add your files under the appropriate folder  
3. Use `/Templates/Test-Submission.md` to format your submission  
4. Submit a Pull Request  
5. Maintainers and community reviewers will evaluate  
6. Contributions will be tagged as “Community Validated” or “Officially Verified”

## Review Workflow
- Community reviews via comments and reactions  
- Maintainers check formatting, structure, and compliance  
- Valid contributions are merged into the main library  

## Compliance Rules
- No patient data (no PHI)  
- No confidential manufacturer firmware  
- All contributions must comply with CC BY-SA 4.0  
- Any clinical use must follow ISO 10993 / FDA / regulatory guidelines  

## Thank You
Your contributions help the entire dental 3D printing community.
